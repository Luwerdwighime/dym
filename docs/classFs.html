◆ mkdir_p()

void Fs::mkdir_p ( std::string path)
static

Создаёт папку (целый путь), если её ещё нет

Функция юзает вызов системной проги "mkdir -p", чтобы без гемора создавать весь путь целиком со всеми родительскими папками. Ошибок не возвращает, сразу падает в std::exit(ERR_DIR)

Аргументы
pathПуть до папки
